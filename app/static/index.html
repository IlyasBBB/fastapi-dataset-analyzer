API Endpoints

Dataset Management
POST /datasets/ - Upload dataset
GET /datasets/ - List datasets
GET /datasets/{id}/ - Get info
DELETE /datasets/{id}/ - Delete dataset
Analysis Endpoints
GET /datasets/{id}/stats/ - Get statistics
GET /datasets/{id}/plot/ - Generate plots
GET /datasets/{id}/excel/ - Export to Excel

Command Line Interface

Available Commands
python client.py list - List all datasets
python client.py upload <file> - Upload dataset
python client.py info <id> - Get dataset info
python client.py delete <id> - Delete dataset
python client.py export <id> - Export to Excel
python client.py stats <id> - Get statistics
python client.py plot <id> - Generate plots
